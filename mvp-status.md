## Current limitations

This is a confirmed executable foundation MVP. It is not yet a complete end-user VS Code replacement.

The next implementation work must address:

- live Code OSS source checkout/build rather than metadata artifacts only;
- real remote-agent upload/start over SSH;
- live Code OSS remote filesystem/watch channels;
- raw terminal lifecycle and real PTY process integration;
- real Code OSS extension-host activation;
- CI execution and retained evidence artifacts;
- closing or splitting the existing GitHub issues once full acceptance criteria are met.
