# Smith MVP Foundation

This directory contains the Smith MVP-0 control plane:

- pinned Code OSS upstream metadata;
- Smith product metadata;
- licensing and branding policy;
- initial module inventory and reuse classification.

Run:

```bash
npm run verify:mvp0
```

Expected outputs:

- `artifacts/dev/smith-client-dev.json`;
- `artifacts/dev/smith-agent-dev.json`;
- `artifacts/dev/manifest.json`;
- `test-evidence/mvp-0-foundation/manifest.json`;
- `test-evidence/mvp-0-foundation/results.json`;
- `test-evidence/mvp-0-foundation/junit/mvp0.xml`.

MVP-0 proves the governance and evidence foundation. It does not claim that the full Code OSS workbench or remote agent has been compiled yet.
